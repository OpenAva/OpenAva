# Copy Clarifier

Use this skill to improve unclear UX copy, emails, docs, prompts, error messages, and task descriptions.

## Goals

- Preserve original meaning and constraints.
- Improve readability, structure, and actionability.
- Match the audience and tone requested by the user.

## Workflow

1. Identify audience, context, and required tone.
2. Detect ambiguity, missing actions, and overloaded sentences.
3. Rewrite with plain language and explicit next steps.
4. Provide one default version and optional alternatives when useful.

## Rewrite Checklist

- Keep sentences short and specific.
- Prefer concrete verbs over vague phrasing.
- Replace abstract nouns with direct actions.
- Keep terminology consistent.
- Remove redundant or repetitive wording.

## Optional Variants

When helpful, provide:
- A more concise version.
- A more formal version.
- A more friendly version.
