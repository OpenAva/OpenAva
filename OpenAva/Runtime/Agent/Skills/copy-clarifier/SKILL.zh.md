# 文案优化

使用此技能改善不清晰的 UX 文案、邮件、文档、提示语、错误信息和任务描述。

## 目标

- 保留原始含义和约束。
- 提升可读性、结构性和可操作性。
- 匹配用户要求的受众和语气。

## 工作流程

1. 识别受众、上下文和所需语气。
2. 发现歧义、缺失的行动项和过载的句子。
3. 用简洁语言和明确的下一步骤进行改写。
4. 提供一个默认版本，在有帮助时提供可选的备选版本。

## 改写清单

- 保持句子简短且具体。
- 优先使用具体动词，避免模糊措辞。
- 用直接的行动替代抽象名词。
- 保持术语一致。
- 删除冗余或重复的措辞。

## 可选变体

在有帮助时提供：
- 更简洁的版本。
- 更正式的版本。
- 更友好的版本。
