# 社交图文卡片

当用户希望把文字内容直接生成为可发布图片（不是文生图）时，使用该技能。

## 能力范围

- 长文本自动分页，生成多张图片
- 字体、间距、留白和段落排版更易读
- 默认笔记卡片风格（类似备忘录阅读体验）
- 支持小红书、Twitter/X 等社交场景输出

## 使用工具

调用本地工具 `text_to_social_images`。

必填参数：

- `text`：要渲染的完整文本

常用可选参数：

- `title`：图片顶部标题
- `theme`：`notes` | `dark`
- `width`：自定义图片宽度（像素）
- `aspectRatio`：如 `3:4`、`4:5`、`16:9`
- `maxPages`：最多生成页数

## 推荐流程

1. 对用户文本做轻量清洗（只修复明显断行问题，不改原意）。
2. 选择主题：
   - `notes`：温和、简洁、阅读优先
   - `dark`：高对比、夜间风格
3. 调用 `text_to_social_images` 生成图片。
4. 明确返回页数和图片路径，方便用户直接发布。

## 约束

- 不得杜撰信息，不改变核心语义。
- 排版以可读性为第一优先，避免花哨装饰。
- 若内容超出 `maxPages`，需明确提示已截断，并建议提高页数上限。
