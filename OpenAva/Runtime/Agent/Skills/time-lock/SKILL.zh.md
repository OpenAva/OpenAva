# 时间锁定

当用户说出"提醒我"、"在周五之前"、"跟进一下"或"不要让我错过这件事"时，使用此技能。

## 核心原则

将意图转化为可靠的系统级触发器，确保重要承诺不会遗失在聊天记录中。

## 工作流程

1. 提取承诺、截止日期和完成条件。
2. 若时间不明确，提出一个精准的后续问题。
3. 选择执行路径：
   - `reminders_add`：用于灵活任务。
   - `calendar_add`：用于固定时间事件。
4. 添加可选的加强措施：
   - `system_notify`：用于即时确认。
   - `cron`：用于重复提醒。
5. 汇总已创建的内容（标题、截止时间、渠道）。

## 安全规则

- 当用户意图不明确时，不要以猜测的日期创建条目。
- 不静默地覆盖现有计划。
- 在最终回复中确认时区假设。
